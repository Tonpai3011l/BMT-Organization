# BMT Organization Bot

ระบบลงทะเบียน Discord bot พื้นฐานที่มาพร้อมกับระบบให้ยศ (Role) อัตโนมัติและระบบบันทึก Log การใช้งาน

## คุณสมบัติ (Features)
- **ระบบลงทะเบียน**: รองรับการกรอกชื่อผ่าน Modal (หน้าต่างป๊อปอัพ)
- **ให้ยศอัตโนมัติ**: มอบยศที่กำหนดให้ทันทีหลังลงทะเบียนสำเร็จ
- **ระบบบันทึก Log**: เก็บประวัติการลงทะเบียน (ชื่อที่กรอก, ผู้ใช้, สถานะการให้ยศ) ไปยังห้องที่กำหนด
- **ความคงทนของข้อมูล (Persistence)**:
    - ปุ่มลงทะเบียนจะยังใช้งานได้แม้บอทจะรีสตาร์ท
    - การตั้งค่าต่างๆ จะถูกเก็บไว้ในไไฟล์ `config.json`

## สิ่งที่ต้องมี (Prerequisites)
- [Python 3.8+](https://www.python.org/)
- ไลบรารีที่จำเป็น:
  ```bash
  pip install discord.py python-dotenv
  ```

## การติดตั้ง (Setup)

1.  สร้างไฟล์ `.env` ในโฟลเดอร์หลัก และใส่ Bot Token ของคุณ:
    ```env
    DISCORD_TOKEN=your_token_here
    ```
2.  รันบอทครั้งแรกเพื่อ Sync คำสั่ง Slash Command:
    ```bash
    python main.py
    ```

## วิธีการใช้งาน (Usage)

### 1. ตั้งค่าห้อง Log
ใช้คำสั่ง `/setlogs` เพื่อเลือกห้องที่จะให้บอทส่งข้อมูลการลงทะเบียนไปเก็บไว้:
```text
/setlogs channel:#ชื่อห้อง
```

### 2. ตั้งค่ายศเริ่มต้น
ใช้คำสั่ง `/set_verify_role` เพื่อเลือกยศที่จะให้ผู้ใช้เมื่อลงทะเบียนสำเร็จ:
```text
/set_verify_role role:@ชื่อยศ
```

### 3. ติดตั้งปุ่มลงทะเบียน
ใช้คำสั่ง `/setup` เพื่อส่งปุ่มลงทะเบียนไปยังห้องที่ต้องการ:
```text
/setup channel:#ชื่อห้อง
```

## โครงสร้างไฟล์ (File Structure)
- `main.py`: โค้ดหลักของบอท
- `config.json`: ไฟล์เก็บค่าคอนฟิก (Log channel, Role ID)
- `.env`: ไฟล์เก็บ Token (ความลับ)
- `README.md`: เอกสารแนะนำการใช้งาน

## การนำขึ้น Host Server (Deployment)

เพื่อให้บอททำงานออนไลน์ได้ตลอด 24 ชั่วโมง แนะนำให้ใช้เครื่องมือจัดการโปรเซส เช่น **PM2**:

1.  **ติดตั้ง PM2**:
    ```bash
    npm install pm2 -g
    ```
2.  **เริ่มการทำงานบอท**:
    ```bash
    pm2 start main.py --name "bmt-bot" --interpreter python3
    ```
3.  **ตั้งค่าให้เปิดอัตโนมัติเมื่อรีสตาร์ทเครื่อง**:
    ```bash
    pm2 save
    pm2 startup
    ```

> [!IMPORTANT]
> อย่าลืมอัปโหลดไฟล์ `.env` และ `config.json` ไปยังเซิร์ฟเวอร์ด้วยตนเอง เนื่องจากไฟล์เหล่านี้ถูกระบุไว้ใน `.gitignore` เพื่อความปลอดภัย
